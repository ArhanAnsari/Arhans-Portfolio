## ⚡ Quick Start

Clone and run locally for development:
```bash
git clone https://github.com/ArhanAnsari/Arhans-Portfolio.git
cd Arhans-Portfolio
npm install
npm run dev
```

Edit `App.jsx` (or use `.tsx` for TypeScript projects) and enjoy live hot reloading with Vite!
